#  TBalLom
### 소개
다양한 배트로 T볼을 체험할 수 있는 웹서비스

#### 📌주요기능
- 게임 : 키보드 방향키를 사용하여 공을 타격, 점수 얻는 게임
- 랭킹 : 얻은 점수로 랭킹 순위 보여주는 기능
- 구매 : 받은 포인트로 배트 구매 기능

#### 🛠️사용언어/툴
- Django, HTML5, CSS3, JavaScript/PyCharm
